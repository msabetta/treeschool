# Treeschool

Benvenuto in **Treeschool**! Questo repository raccoglie una serie di **esercitazioni e progetti realizzati durante un corso per Front-End Developer**. L'obiettivo è mettere in pratica le competenze di sviluppo web, partendo da esercizi semplici fino a progetti più complessi.

## 📁 Struttura del Progetto

*   **`form-registrazione/`**: Un progetto dedicato alla creazione di un modulo di registrazione. All'interno troverai i file HTML, CSS e JavaScript necessari per realizzare un form interattivo e funzionale, con validazione dei dati e un layout accattivante.

## 🚀 Come Visualizzare i Progetti

Ogni cartella contiene un progetto a sé stante. Per visualizzarli:

1.  **Clona il repository** sul tuo computer:
    ```bash
    git clone https://github.com/msabetta/treeschool.git
    ```
2.  **Naviga nella cartella del progetto** che ti interessa, ad esempio:
    ```bash
    cd treeschool/form-registrazione
    ```
3.  **Apri il file `index.html`** (o il file principale) nel tuo browser web preferito. Puoi farlo semplicemente con un doppio clic sul file o usando le funzionalità del tuo editor di codice.

## 🛠️ Progetti Inclusi

### 1. Modulo di Registrazione (`/form-registrazione`)
Questo progetto è un esercizio fondamentale per imparare a gestire i moduli HTML. Include tipicamente:
*   **HTML**: Struttura del modulo con campi per nome, email, password, ecc.
*   **CSS**: Stile e layout per rendere il modulo graficamente gradevole e responsive.
*   **JavaScript**: Logica per la validazione dei dati inseriti dall'utente (es. controllare che la password sia abbastanza lunga, che l'email sia in un formato valido) e fornire feedback immediato.

## 🎯 Obiettivi del Corso

Attraverso questi esercizi, l'autore (e chiunque voglia esplorare il repository) si esercita con:
*   I fondamenti di **HTML5** e la semantica delle pagine web.
*   Le tecniche di stilizzazione con **CSS3** (flexbox, grid, animazioni).
*   La programmazione lato client con **JavaScript** (manipolazione del DOM, gestione eventi, validazione form).

## 🤝 Contributi e Prossimi Passi

Questo repository è in continua evoluzione man mano che il corso procede. Nei prossimi commit potrebbero essere aggiunte nuove cartelle con esercizi su:
